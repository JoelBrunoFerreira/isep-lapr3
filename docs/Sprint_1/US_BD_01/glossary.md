# Glossário

**Termos, Expressões e Siglas (TES) devem ser organizados em ordem alfabética.**

| **_TES_**      | **_Descrição_**                                                                                                                                                                                                                                                                                                                                                            |
|:---------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Adubos         | Qualquer tipo de substância aplicada ao solo ou tecidos vegetais para fornecer um ou mais nutrientes essenciais ao crescimento das plantas.                                                                                                                                                                                                                                |
| Adubos orgânicos | Adubos ricos em nutrientes e minerais criados a partir da decomposição de materiais orgânicos como restos de comida, resíduos de plantas e fezes de animais.                                                                                                                                                                                                               |
| Adubação verde | Prática de incorporar plantas leguminosas no solo para enriquecê-lo nutricionalmente com nitrogénio.                                                                                                                                                                                                                                                                       |
| Agricultura     | Prática de cultivar plantas e criar gado.                                                                                                                         |
| Agricultura biológica | Método de produção alimentar que se baseia na sustentabilidade e responsabilidade agroecológica e social.                                                                                                                                                                                                                                                                  |
| Biodiversidade  | Biodiversidade, ou diversidade biológica, pode ser definida como a variabilidade entre os seres vivos de todas as origens, terrestres, marinhas e de outros ecossistemas aquáticos, bem como os complexos ecológicos dos quais fazem parte.                                                                                                                                |
| Cadeia alimentar | Sequência de seres vivos na qual um organismo serve de alimento para o outro.                                                                                                                                                                                                             |
| Caderno de campo | Documento formal e obrigatório em contexto de agricultura biológica que permite registar as operações agrícolas relevantes (fertilizações, dados recolhidos pelos sensores metereológicos, registo de colheitas, regas).                                                                                                                                                   |
| Ciclo de cultura | Período que ocorre desde a sementeira até à colheita.                                                                                                                                                                                                                                                                                                                      |
| Cliente        | Pessoa que encomenda e consome produtos agrícolas, distribuídos na forma de cabazes.                                                                                                                                                                                                                                                                                       |
| Colheita       | Conjunto de produtos agrícolas colhidos num determinado período de tempo.                                                                                                                                                                                                                                                                                                  |
| Condutor       | Pessoa responsável por recolher os cabazes na exploração agrícola e depositá-las nos Hubs de distribuição.                                                                                                                                                                                                                                                                 |
| Controlador (sistema de rega) | Equipamento responsável pela gestão da operação do sistema de rega para um conjunto específico de parcelas.                                                                                                                                                                                                                                                                |
| Cultura        | Espécie vegetal cultivada na agricultura com o objetivo de produzir produtos agrícolas para consumo humano/animal (por exemplo, flores, frutos, cereais, hortaliças, forragens, fibras) ou para produzir adubação verde.  |
| Ecossistema    | Conjunto formado pelos seres vivos de um ambiente e pelas relações estabelecidas entre eles.                                                                                                                                                                                                                                                              |
| Ecossistema agrícola (agroecossistema) | Local de produção agrícola.                                                                                                                                                                                                                                                                                                                                                |
| Emparelhamento do solo | Processo de determinar a compatibilidade entre diferentes tipos de solo e culturas agrícolas, levando em consideração as características do solo e as necessidades da cultura.                                                                                                                                                                                             |
| Estação meteorológica | Conjunto de sensores que medem variáveis atmosféricas (vento, humidade, radiação, pressão). Fornecem monitorização adequada das condições climáticas e, consequentemente, um uso mais sustentável dos recursos (água, fatores de produção).                                                                                                                                 |
| Fator de Produção | Produtos que são aplicados no solo ou nas plantas de forma a melhorar e nutrir os mesmos, prevenir doenças, corrigir desequilíbrios e combater pragas e doenças. Podem ser classificados como fertilizantes, adubos, corretivos ou produtos fitofármacos.                                                                       |
| Fertilização   | Ato de adicionar nutrientes essenciais ao solo ou às plantas para promover o crescimento saudável das culturas.                                                                                                                                                                                                                                                            |
| Gestor Agrícola | Pessoa que gere culturas nas parcelas, realiza diferentes operações e regista-as no Caderno de Campo.                                                                                                                                                                                                                                                           |
| Gestor de Distribuição | Pessoa que gere o processo de recolha e transporte de produtos agrícolas entre explorações, Hubs e posterior recolha pelos clientes.                                                                                                                                                                                                                                   |
| Instalação agrícola | Quinta composta por um conjunto de parcelas e edifícios.                                                                                                                                                                                                                                                                                                                   |
| Parcela Agrícola | Uma parcela, também designada por campo, é caracterizada por uma designação, área (ha) e culturas (macieira, pereira, trigo, feijão), que variam ao longo do tempo.                                                                                                                                                                                                        |
| Plano de rega e fertilização | Plano (tipicamente anual) das necessidades de rega, correções minerais e fertilizações a serem realizadas ao longo da época agrícola.                                                                                                                                                                                                                                      |
| Plantação      | Ação de estabelecer um grupo de plantas, muitas vezes em linhas ou fileiras, com o propósito de cultivar uma cultura específica.                                                                                                                                                                                                                                           |
| Plantação permanente | Cultivo de espécies vegetais que permanecem no local por várias estações ou anos, como árvores frutíferas, videiras, etc.                                                                                                                                                                                                                                                  |
| Plantação temporária | Cultivo de culturas que são semeadas e colhidas dentro de uma única temporada de crescimento, como milho, trigo, legumes, etc.                                                                                                                                                                                                                                             |
| Polinização    | Processo pelo qual o pólen de uma flor é transferido para o estigma de outra flor, permitindo a fertilização e a produção de sementes e frutos.                                                                                                                                                                                                                            |
| Rega           | Ato de verter um líquido sobre uma determinada superfície com o intuito de lhe proporcionar algum benefício. A distribuição pode ser feita por aspersão, gotejamento ou pulverização.                                                                                                                                                                                      |
| Rega bombeada  | Rega onde se recorre à utilização de bombas hidráulicas.                                                                                                                                                                                                                                                                                                                   |
| Rega por gravidade | Rega realizada com a existência de um desnível entre a fonte e as culturas a serem regadas.                                                                                                                                                                                                                                                                                |
| Rotação de culturas | Prática agrícola que envolve alternar as culturas plantadas em uma determinada área ao longo das estações para melhorar a saúde do solo e reduzir o risco de pragas e doenças.                                                                                                                                                                                             |
| Sensor de humidade do solo | Dispositivo que mede o teor de água presente no solo para auxiliar na gestão da rega.                                                                                                                                                                                                                                                                                      |
| Sensor de pluviosidade | Dispositivo que mede a quantidade de chuva que cai numa determinada área.                                                                                                                                                                                                                                                                                                  |
| Sensor de temperatura, e pressão atmosférica | Sensor que mede a temperatura do ar na instalação agrícola.                                                                                                                                                                                                                  |
| Sensor de humidade do ar | Sensor que mede a humidade do ar na instalação agrícola. 
| Sensor de pressão atmosférica | Sensor que mede a pressão atmosférica na instalação agrícola. |
| Sensor de temperatura do solo | Dispositivo que mede a temperatura do solo em diferentes profundidades.                                                                                                                                                                                                                                                                                                    |
| Sensor de velocidade do vento | Dispositivo que mede a velocidade e direção do vento, fornecendo informações sobre as condições climáticas.                                                                                                                                                                                                                                                                          |
| Sistema de rega  | Conjunto de equipamentos capaz de fazer chegar água ou soluções aquosas contendo fatores de produção.                                                                                                                                                                                                                                                                      |
